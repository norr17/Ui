# Example

Use `Example.lua` as the executable showcase.

It demonstrates:
- bootstrap and lifecycle
- tabs and sections
- toggles, sliders, dropdowns, textboxes, keybinds, color picker
- radio group and progress bar
- config save/load/autoload
- optional `KojoExtended` widgets
- watermark and FPS counter

Run the file locally with `readfile`, or set `getgenv().KojoRepo` and let it fetch from raw URLs.
