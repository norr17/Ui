# Example

`Example.lua` is an executor-ready consumer script.

Run it directly:

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/norr17/Ui/main/Example.lua"))()
```

What it demonstrates:
- remote loading from raw GitHub URLs
- `KojoLib.lua`, `KojoExtended.lua`, `SaveManager.lua`, `ThemeManager.lua`
- tabs, groupboxes, tabboxes, key tab
- toggles, checkbox, slider, dropdown, input, keybind, key picker, color picker
- `Library.Options` and `Library.Toggles`
- `OnChanged` and `SetValue`
- config UI and autoload
- theme preset UI
